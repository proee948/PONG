classic pong game made in C, playable entirely in terminal.

## Dependencies
Debian/Ubuntu:

`sudo apt install libncurses-dev libcdk5-dev`

Fedora:

`sudo dnf install ncurses-devel cdk-devel`

Build & Run

`make`

then

`./pong`

Controls

|Player 1|Player 2|
|--------|--------|
|W — up  |↑ — up  |
|S — down|↓ — down|
